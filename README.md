# Introduccion 

This project is payroll sysmtem based in a goverment version.

## UI Stack
- React + Typescript
- AntD
- Redux
- Redux-saga
- Axios

## Installation

```bash
$ npm install
```
## Scripts

```bash
$ npm run start
```
